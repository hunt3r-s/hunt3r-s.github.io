GNU Typist, why this 30 year old typing tutor is unmatched

Table of Contents

  • 1. If it Ain’t Broke…
  • 2. Write your own lessons
  • 3. Why use anything else?

GNU Typist is a typing tutor from the GNU Project. It provides a command line interface with guided typing lessons. It also provides a method of constructing your own lessons. It was created by Jim Meyering, and is part of the GNU project, a well-known. ongoing effort to create a completely free and open-source operating system. Despite its age, GNU Typist is still unmatched as a typing tutor, no modern solutions match the simplicity, portability, and effectiveness of gtypist’s lessons.

1. If it Ain’t Broke…

I have recently come to realize that touch typing is an incredibly valuable skill, one that I lack. As a result, I have been exploring a rabbit hole of touch typing tutors, both online and desktop based. Some of the services I have tried include

  • keybr.com
  • 10fastfingers
  • and of course, typeracer. While all of these typing tutors and games have their merits, I have not found any useful features present in these services, that are absent in GNU Typist.

In fact, many of these services lack the element that sets gtypist apart, guided lessons. I have not found another typing tutor that provides specific instructions on where to place each finger. This is somewhat surprising to me, since it is the single most important aspect of touch typing. For example, if you have never touch typed before, you may like to start off with the basic QWERTY course. It teaches touch typing in a proper, starting with the home row and gradually introducing new keys and finger combinations. The program provides speed drills to practice each new skill before moving on, ensuring you have a solid foundation before progressing. No frills.

This course took me from 10WPM to 30+WPM within a week of learning to touch type. While other typing services provide fancy GUIs and fun stuff, gtypist focuses on what is important, teaching the user to type accurately and quickly.

2. Write your own lessons

Another massive advantage of gtypist is the ability to write custom lessons. This opens the door to community developed lessons. Gtypist can be run with an optional argument, which points a custom .typ file which contains lessons. Lessons are written in a light markup language (I’m not sure if it has a name). Within the custom files, you can specify lesson information, such as instructions for typing, and custom text for the drills. The GNU Typist documentation provides detailed information for constructing custom lessons. For example, I wrote a short Python script to generate lessons from the works of Shakespeare. 3. Why use anything else?

While I may not feel the need to explore any other typing services, I can see a few reasons why gtypist might be insufficient for some learners.

  • Typing Games - If you are someone who likes to compete against other people, or practice typing in a game-like environment, gtypist may not be for you. It can get a bit lonely typing to yourself.
  • Scared of the command line - If you don’t like the command line, or just prefer GUIs, gtypist may not be for you. Although, it only requires one command to run, so it shouldn’t be too intimidating
  • You want in depth statistics - If you are someone who want to deeply analyze your typing habits, GNU Typist may not be your best choice. It does, however, keep track of a few basic stats such as WPM history and accuracy.
